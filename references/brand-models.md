## 1. 定位三角形

### 核心公式
```
有效定位 = 目标客群 × 竞争差异 × 核心价值

三边必须同时成立：
- 客户需要（要有人买）
- 竞争差异（要有不同）
- 自身能做（要有能力）
```

### 三个维度检验

| 维度 | 检验问题 | 失败信号 |
|------|---------|---------|
| 目标客群 | 足够具体吗？足够大吗？ | "所有人都是客户" |
| 竞争差异 | 差异可感知？可验证？ | "我们也差不多" |
| 核心价值 | 能兑现吗？可持续吗？ | 承诺与体验脱节 |

### 定位三角画布
```
           目标客群
              △
             ╱ ╲
            ╱   ╲
           ╱     ╲
          ╱       ╲
         ╱  品牌定位  ╲
        ╱   一句话定位  ╲
       ╱               ╲
      ▼─────────────────▼
    核心价值 ←─────────→ 竞争差异
```

## 2. Kapferer 品牌识别棱镜

### 六维度品牌架构
```
        物理层面                    个性层面
      ┌─────────┐                ┌─────────┐
      │ Physique│                │Personality│
      │  产品形体 │                │ 品牌个性  │
      └────┬────┘                └────┬────┘
           │                          │
    ┌──────┴──────────────────────────┴──────┐
    │               品牌核心                  │
    │           Brand Core Value             │
    └──────┬──────────────────────────┬──────┘
           │                          │
      ┌────┴────┐                ┌────┴────┐
      │ Culture │                │Relationship│
      │ 品牌文化 │                │ 客户关系  │
      └─────────┘                └─────────┘
           │                          │
    ┌──────┴──────┐            ┌──────┴──────┐
    │Reflection   │            │Self-image   │
    │ 客户镜像     │            │ 理想自我    │
    └─────────────┘            └─────────────┘
```

### 六维度定义

| 维度 | 核心问题 | 描述要点 |
|------|---------|---------|
| Physique | 产品长什么样？ | 外观、功能、特征 |
| Personality | 品牌像谁？ | 性格、语调、代言人人设 |
| Culture | 品牌信仰什么？ | 价值观、使命、源起故事 |
| Relationship | 与客户什么关系？ | 朋友、导师、伙伴、专家 |
| Reflection | 客户怎么看我们？ | 使用者形象、社会标签 |
| Self-image | 客户想成为什么？ | 理想自我、渴望身份 |

## 3. Aaker 品牌资产模型

### 品牌资产五维度
```
品牌资产（Brand Equity）
    │
    ├─ 1. 品牌认知（Awareness）
    │       └─ 回忆率、识别率、第一提及率
    │
    ├─ 2. 品牌联想（Associations）
    │       └─ 功能属性、情感属性、符号属性
    │
    ├─ 3. 感知质量（Perceived Quality）
    │       └─ 品质评价、可靠性、精致度
    │
    ├─ 4. 品牌忠诚（Loyalty）
    │       └─ 复购率、推荐率、钱包份额
    │
    └─ 5. 其他资产（Other Assets）
            └─ 专利、商标、渠道关系
```

### 品牌健康诊断指标

| 指标 | 测量方法 | 健康标准 |
|------|---------|---------|
| 认知度 | 无提示回忆率 | >60% |
| 联想数 | 品牌联想词数 | >5个正向联想 |
| 质量感知 | 10分量表 | >7分 |
| 忠诚度 | NPS净推荐值 | >30 |

## 4. 价值主张画布

### 双面匹配框架
```
┌─────────────────────────────────────┐
│           价值主张                   │
├─────────────────┬───────────────────┤
│  产品与服务     │   痛点缓释器        │
│  （What）       │   （Pain Relievers）│
│                 │                   │
│  增益创造器     │                   │
│  （Gain Creators）│                   │
└─────────────────┴───────────────────┘
              ↑ 匹配 ↓
┌─────────────────┬───────────────────┤
│   客户任务      │     痛点          │
│  （Jobs）       │   （Pains）        │
│                 │                   │
│   收益         │                   │
│  （Gains）      │                   │
└─────────────────┴───────────────────┘
```

### 完整检验清单
```
客户侧：
□ 客户要完成什么任务（Jobs）？
□ 什么让客户痛苦（Pains）？
□ 客户期望什么收益（Gains）？

产品侧：
□ 我们提供什么产品服务？
□ 如何缓解客户痛点？
□ 如何创造额外收益？

匹配检验：
□ 痛点缓释器对应真实痛点吗？
□ 增益创造器对应期望收益吗？
□ 价值主张独特且难以复制吗？
```

## 5. PMM 框架

### 核心三角
```
P（Product）产品 × M（Market）市场 × M（Message）信息

营销效果 = P×M×M 三者匹配度
```

### 三要素诊断

| 要素 | 检验问题 | 常见错配 |
|------|---------|---------|
| Product | 产品是否解决了真实问题？ | 自嗨产品、需求伪命题 |
| Market | 目标市场足够大且可达？ | 市场太小、渠道不通 |
| Message | 信息是否传达核心价值？ | 信息模糊、传播失真 |

### PMM 匹配度评估
```
【PMM诊断】
产品-市场匹配度：__/10
市场-信息匹配度：__/10
产品-信息匹配度：__/10

错配诊断：
□ P-M错配：产品好但市场小 → 调整目标市场或产品线
□ M-M错配：营销信息偏离市场需求 → 重构信息策略
□ P-M错配：宣传与产品不符 → 调整产品或诚实传达
```

## 6. 品牌护城河

### 巴菲特护城河五要素
```
1. 无形资产：品牌、专利、牌照
2. 转换成本：用户替代的时间/金钱/风险成本
3. 网络效应：用户越多价值越高
4. 成本优势：规模效应/独特资源
5. 规模优势：市场份额、渠道覆盖
```

### 护城河强度评估

| 护城河类型 | 强度 | 案例 |
|-----------|------|------|
| 网络效应 | ⭐⭐⭐⭐⭐ | 微信、抖音 |
| 转换成本 | ⭐⭐⭐⭐ | iOS生态、企业软件 |
| 品牌溢价 | ⭐⭐⭐⭐ | 苹果、茅台 |
| 成本优势 | ⭐⭐⭐ | 拼多多、Costco |
| 规模效应 | ⭐⭐⭐ | 快消品、制造业 |

### 品牌护城河自检
```
□ 用户是否愿意为你支付溢价？
□ 用户离开的成本有多高？
□ 新进入者需要多长时间追赶？
□ 护城河是否因技术/趋势变化而弱化？
```

## 7. 品牌原型理论

### 12 种品牌原型

| 原型 | 特征 | 品牌 | 核心信息 |
|------|------|------|---------|
| 英雄 | 勇敢、征服 | Nike | Just Do It |
| 反叛者 | 打破规则 | Apple (早期) | Think Different |
| 魔术师 | 转化、超越 | Disney | Make Magic |
| 凡人 | 接地气、平等 | IKEA | 为大众创造美好生活 |
| 情人 | 亲密、感性 | Chanel | 诱惑、魅力 |
| 小丑 | 快乐、幽默 | M&M's | 只融在口 |
| 智者 | 真理、知识 | Google | 组织世界信息 |
| 探索者 | 自由、发现 | Jeep | 没有路也能去 |
| 统治者 | 控制、权力 | Mercedes-Benz | 巅峰驾驭 |
| 照顾者 | 关怀、服务 | 强生 | 关爱每一家庭 |
| 创造者 | 创意、表达 | Lego | 激发创造力 |
| 天真者 | 纯真、乐观 | Coca-Cola | 开启快乐 |

### 品牌原型选择
```
选择标准：
1. 与目标用户价值观共鸣
2. 与企业真实文化匹配
3. 与竞品原型差异明显
4. 可持续，不易疲劳

避免：
- 频繁切换原型（混乱）
- 原型与产品不符（撕裂）
- 模仿竞品原型（同质）
```
