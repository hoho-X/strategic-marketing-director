## 一、13字顶级心法 ✅ 已补全

来源：豆瓣书评直接引用 + 书籍目录结构交叉验证。

> **"戒除自我，对'你'说、私下说、对'你'好"**

### 逐字拆解

| 字词 | 含义 | 操作指引 |
|------|------|---------|
| **戒除自我** | 放下"我想说什么"，切换为"用户想听什么" | 写之前问：如果我是用户，这句话对我有价值吗？ |
| **对"你"说** | 文案像和具体的某个人对话，不是面向群体喊话 | 把"各位朋友"改成"你"，每段只对一个人说话 |
| **私下说** | 制造一对一的私密沟通感，信息有了温度 | 用"偷偷告诉你""只有你知道"这类亲密信号 |
| **对"你"好** | 起心动念必须是为对方创造价值，不夹带私货 | 写完检查：这段话到底帮了用户什么？ |

### 如何应用这13字

```
写之前 → 戒除自我：我不是在展示文采，我是在帮一个人
写之中 → 对"你"说：像给朋友发微信，不是像在台上演讲
写之中 → 私下说：让信息变成"我们之间的小秘密"
写之后 → 对"你"好：念一遍，你是用户，你觉得这对你有用吗？
```

## 二、5个戒除自嗨的方法 ✅ 已补全

基于书籍简介和彭芳公开分享内容整理：

| # | 方法 | 具体做法 |
|---|------|---------|
| 1 | **用"你"代替"我们"** | 不是在说你多厉害，是在说你帮用户解决什么问题 |
| 2 | **写痛点，不写卖点** | 用户在痛的时候才会想解决——先写痛点画面，再给方案 |
| 3 | **写具体，不写抽象** | "提升效率"→"以前花3小时，现在30分钟" |
| 4 | **写感受，不写参数** | 用户不为参数买单，为"用了之后的感受"买单 |
| 5 | **写故事，不写道理** | 用真实客户案例代替产品说明书 |
