# 营销传播模型

> 营销组合 / 市场策略 / 传播路径 / 整合营销框架

---

## 1. 4P 营销组合

### 经典四要素
```
Product（产品）+ Price（价格）+ Place（渠道）+ Promotion（促销）
```

| 要素 | 关键决策 | 常见策略 |
|------|---------|---------|
| Product | 产品功能、质量、包装、品牌 | 差异化、产品线延伸 |
| Price | 定价策略、折扣政策 | 撇脂/渗透/心理定价 |
| Place | 渠道选择、终端管理 | 直销/分销/全渠道 |
| Promotion | 广告、公关、促销、人员推销 | 整合营销传播 |

### 4P 到 4C 演进
```
Product → Customer Solution（客户解决方案）
Price   → Customer Cost（客户成本）
Place   → Convenience（便利性）
Promotion → Communication（沟通）
```

## 2. STP 策略

### 三步流程
```
Segmentation（细分）→ Targeting（选择）→ Positioning（定位）
```

### 市场细分维度

| 维度 | B2B | B2C |
|------|-----|-----|
| 地理 | 区域、城市等级 | 城市、气候、人口密度 |
| 人口 | 企业规模、行业 | 年龄、性别、收入、教育 |
| 行为 | 购买频率、使用场景 | 购买频率、忠诚度、使用场景 |
| 心理 | 企业文化、价值观 | 生活方式、价值观、个性 |

### 目标市场选择策略

| 策略 | 特点 | 适用场景 |
|------|------|---------|
| 无差异化 | 同一产品同一策略 | 需求高度同质化 |
| 差异化 | 多产品多策略 | 多个细分市场 |
| 集中化 | 聚焦单一细分市场 | 资源有限、利基市场 |
| 微观营销 | 一对一定制 | 高价值客户、技术支持 |

### 定位策略矩阵
```
          差异化程度
            高    低
        ┌──────┬──────┐
   高   │产品差异│服务差异│
竞争强度├──────┼──────┤
   低   │价格领先│利基定位│
        └──────┴──────┘
```

## 3. AIDA 模型

### 注意力漏斗
```
Attention（注意）→ Interest（兴趣）→ Desire（欲望）→ Action（行动）
```

| 阶段 | 目标 | 策略 | 指标 |
|------|------|------|------|
| Attention | 吸引注意力 | 震撼视觉、痛点直击、好奇心 | 曝光量、点击率 |
| Interest | 引发兴趣 | 利益点呈现、案例展示 | 停留时长、浏览页数 |
| Desire | 激发欲望 | 社交证明、稀缺性、情感共鸣 | 咨询率、加入购物车 |
| Action | 促成行动 | 限时优惠、降低门槛、信任保障 | 转化率、成交额 |

### AIDA 扩展模型
```
AIDA → AISAS（互联网时代）
Attention → Interest → Search → Action → Share

AIDA → AIDRA（数字营销）
Attention → Interest → Desire → Rationalization → Action
```

## 4. PAS 文案公式

### 三步说服框架
```
Problem（问题）：指出痛点
Agitation（放大）：放大痛苦、制造紧迫感
Solution（方案）：提供解决方案
```

### 应用示例
```
P：你是不是每天加班到深夜，却总觉得效率低下？
A：错过孩子成长、健康亮红灯、升职总轮不到你...
S：时间管理训练营，3周让你每天多出2小时，工作生活两不误
```

### PAS 变体
```
FAB（特性-优势-利益）：
Feature（特性）：产品有什么？
Advantage（优势）：比竞品好在哪？
Benefit（利益）：给用户带来什么？

BAB（Before-After-Bridge）：
Before：使用前的痛苦状态
After：使用后的美好状态
Bridge：产品如何连接两者
```

## 5. GTM 策略（Go-To-Market）

### 市场进入五要素
```
1. 目标客户（Who）：ICP画像、决策者
2. 价值主张（What）：差异化定位
3. 定价策略（How much）：定价模型、折扣策略
4. 销售渠道（Where）：直销/渠道/线上
5. 营销推广（How）：触达策略、内容策略
```

### GTM 路线图模板
```

- 渠道：____
- 关键指标：____

阶段2：成长期（M4-M9）
- 目标：规模化获客
- 渠道：____
- 关键指标：____

阶段3：成熟期（M10+）
- 目标：市场份额巩固
- 渠道：____
- 关键指标：____
```

## 6. ICP 画像模型

### 理想客户画像结构
```
ICP = Firmographics（企业特征）+ Demographics（人口特征）+ Technographics（技术特征）+ Psychographics（心理特征）+ Behavioral（行为特征）
```

### B2B ICP 模板
```
【企业特征】
- 行业：____
- 规模：员工_人，营收_万
- 阶段：初创/成长/成熟
- 地理位置：____

【决策者特征】
- 职位：____
- 痛点优先级：____
- 预算权限：____
- 购买周期：____

【技术特征】
- 现有技术栈：____
- 数字化程度：____
- 集成需求：____

【购买行为】
- 信息获取渠道：____
- 决策流程：____
- 成功案例需求：____
```

### ICP 验证清单
```
□ 是否基于真实客户数据？
□ 是否足够具体可执行？
□ 市场规模是否足够？
□ 是否与销售能力匹配？
□ 是否定期更新验证？
```

## 7. SOSTAC-RACE 模型

### SOSTAC 规划框架
```
S（Situation）：现状分析
O（Objectives）：目标设定
S（Strategy）：策略制定
T（Tactics）：战术执行
A（Action）：行动计划
C（Control）：监控优化
```

### RACE 执行框架
```
Reach（触达）→ Act（互动）→ Convert（转化）→ Engage（维系）
```

| 阶段 | 目标 | 关键动作 | 指标 |
|------|------|---------|------|
| Reach | 扩大曝光 | SEO、广告、内容营销 | 曝光量、流量 |
| Act | 引发互动 | 内容互动、社交分享 | 停留时长、注册率 |
| Convert | 促成转化 | 优惠、信任保障 | 转化率、客单价 |
| Engage | 提升忠诚 | 会员、社群、服务 | 复购率、NPS |

### 整合应用
```
SOSTAC 规划 → RACE 执行 → PDCA 迭代
```

## 8. 整合营销传播（IMC）

### IMC 核心原则
```
一个声音（One Voice）：所有触点传递一致信息
一个形象（One Image）：视觉识别统一
一个定位（One Position）：核心价值一致
```

### 接触点整合矩阵

| 阶段 | 接触点 | 内容类型 | 关键信息 |
|------|--------|---------|---------|
| 认知 | 广告、PR、KOL | 品牌故事、价值主张 | 我们是谁、为什么不同 |
| 兴趣 | 官网、内容、社群 | 产品介绍、案例 | 能解决什么问题 |
| 考虑 | 销售、客服、口碑 | 详细方案、报价 | 为什么选我们 |
| 购买 | 电商、门店、销售 | 促销、保障 | 立即行动的理由 |
| 忠诚 | 售后、会员、社群 | 服务、内容、活动 | 持续选择我们的理由 |

### IMC 一致性检查
```
□ 广告语与产品体验一致？
□ 线上线下视觉统一？
□ 不同渠道传递相同核心信息？
□ 客服话术与品牌调性一致？
□ 危机公关回应符合品牌价值观？
```
