# 战略分析模型

> 宏观环境 / 行业竞争 / 内部能力 / 战略方向分析框架

## 1. 波特五力分析（Porter's Five Forces）

### 核心公式
```
行业吸引力 = f(五力合力)
  ├ 供应商议价能力
  ├ 买方议价能力
  ├ 新进入者威胁
  ├ 替代品威胁
  └ 行业内竞争强度
```

### 评估维度

| 力量 | 高威胁信号 | 低威胁信号 |
|------|-----------|-----------|
| 供应商议价 | 集中度高、切换成本高、无替代 | 供应商分散、标准化原料 |
| 买方议价 | 集中采购、低切换成本、价格敏感 | 分散客户、高差异化 |
| 新进入者 | 低壁垒、低资本要求、无规模效应 | 强壁垒、品牌/专利/网络效应 |
| 替代品 | 功能相近、价格更低、趋势向上 | 无直接替代、切换成本高 |
| 行业竞争 | 增长慢、同质化、退出壁垒高 | 差异化强、增长快、玩家少 |

### 输出模板
```
【五力评估】
行业：____
整体吸引力：高/中/低

| 力量 | 评分(1-5) | 关键判断依据 |
|------|----------|------------|
| 供应商议价 | _ | _ |
| 买方议价 | _ | _ |
| 新进入者 | _ | _ |
| 替代品 | _ | _ |
| 行业竞争 | _ | _ |

关键杠杆点：____
战略建议：____
```

## 2. SWOT 分析

### 核心逻辑
```
SO策略（优势×机会）= 杠杆放大
WO策略（劣势×机会）= 补短板抓机会
ST策略（优势×威胁）= 防御利用
WT策略（劣势×威胁）= 避险退出
```

### 评估矩阵

| | 机会（O） | 威胁（T） |
|---|----------|----------|
| **优势（S）** | SO：如何放大优势抓住机会？ | ST：如何用优势抵御威胁？ |
| **劣势（W）** | WO：如何补短板抓机会？ | WT：如何避免劣势被威胁放大？ |

### 严格规则
- 每个S/W/O/T项必须可量化或可验证
- S和W是**内部**因素，O和T是**外部**因素
- 禁止"既是优势又是机会"的模糊表述
- 每个象限至少产出1条可执行策略

## 4. BCG 矩阵

### 核心公式
```
相对市场份额（横轴）× 市场增长率（纵轴）= 四象限定位
```

| 象限 | 特征 | 策略 | 现金流 |
|------|------|------|--------|
| ⭐ 明星 | 高份额×高增长 | 加大投入，巩固领先 | 净投入大 |
| 💰 现金牛 | 高份额×低增长 | 维持收割，榨取利润 | 净产出大 |
| ❓ 问号 | 低份额×高增长 | 选择性投入或退出 | 净投入大 |
| 🐕 瘦狗 | 低份额×低增长 | 退出或清算 | 微利/亏损 |

### 决策规则
- 现金牛利润 → 喂养明星和有潜力的问号
- 问号只选1-2个重点投入，其余放弃
- 瘦狗无战略价值时坚决退出
- 投资组合健康度 = 明星+现金牛占比 > 60%

## 5. 安索夫矩阵

### 增长路径选择

| | 现有产品 | 新产品 |
|---|---------|--------|
| **现有市场** | 市场渗透（低风险） | 产品开发（中风险） |
| **新市场** | 市场开发（中风险） | 多元化（高风险） |

### 风险-收益评估
```
市场渗透：R≈1x → 增长天花板低
市场开发：R≈2x → 需渠道适配
产品开发：R≈3x → 需研发投入
多元化：  R≈5x+ → 需全新能力
```

### 选择规则
- 优先填满市场渗透空间（最安全）
- 市场开发前验证渠道可达性
- 产品开发基于现有用户未满足需求
- 多元化仅在核心业务饱和时考虑

## 6. 3C 战略三角

### 核心框架
```
战略三角 = Customer（顾客）× Company（自身）× Competitor（竞争者）

战略定位 = 三者交集的差异化空间
```

| 角度 | 关键问题 | 分析工具 |
|------|---------|---------|
| Customer | 目标客户是谁？核心需求？ | ICP画像、需求层次 |
| Company | 我们的核心能力？差异化？ | 价值链分析、资源审计 |
| Competitor | 竞争格局？对手优势？ | 五力分析、竞品矩阵 |

### 战略定位检验
```
□ 是否基于客户真实需求（不是假设）？
□ 是否发挥了自身独特优势（不是模仿）？
□ 是否相对竞争者有差异化（不是同质）？
□ 三者交集是否足够大且可持续？
```

## 7. 蓝海战略

### 核心方法论：战略画布 + 四步动作框架

**四步动作**：
```
消除（Eliminate）：行业惯例中哪些应消除？
减少（Reduce）：哪些应降低到行业标准以下？
增加（Raise）：哪些应提升到行业标准以上？
创造（Create）：哪些行业从未提供应创造？
```

**价值创新公式**：
```
蓝海 = 价值创新 ≠ 技术创新
     = 同时追求差异化 AND 低成本
     = 打破"差异化=高成本"的权衡
```

**六条路径**：
1. 跨越替代行业
2. 跨越战略集团
3. 跨越买方链
4. 跨越互补产品/服务
5. 跨越功能-情感导向
6. 跨越时间趋势

### 输出模板
```
【蓝海战略画布】
行业：____

| 竞争要素 | 行业标准 | 我们 | 消除 | 减少 | 增加 | 创造 |
|---------|---------|------|------|------|------|------|
| _ | _ | _ | ☐ | ☐ | ☐ | ☐ |

价值曲线差异化点：____
非客户群体：____
```
