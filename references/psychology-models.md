# 消费者心理模型

> 行为驱动 / 决策影响 / 用户旅程 / 心理洞察框架

## 1. BJ Fogg 行为模型

### 核心公式
```
Behavior = Motivation × Ability × Trigger
行为 = 动机 × 能力 × 触发器

三者必须同时满足：
- 动机够强
- 能力门槛够低
- 触发器时机对
```

### 三要素优化

| 要素 | 提升策略 | 案例 |
|------|---------|------|
| Motivation | 强化利益、情感共鸣、社会认同 | 限时优惠、稀缺性 |
| Ability | 简化流程、降低门槛、预填充 | 一键购买、自动填充 |
| Trigger | 时机精准、文案吸睛、场景匹配 | 购物车提醒、上新通知 |

### 行为设计检查
```
□ 动机层面：用户为什么要做？（利益/情感/认同）
□ 能力层面：用户能轻松做到吗？（时间/金钱/认知成本）
□ 触发层面：时机和方式对吗？（推送/弹窗/提醒）
```

## 2. COM-B 模型

### 行为改变三要素
```
Capability（能力）+ Opportunity（机会）+ Motivation（动机）→ Behavior（行为）
```

### 诊断框架
```
用户不执行的诊断路径：

问题：转化率低
├─ Capability？
│   ├─ 知道怎么做吗？
│   └─ 有能力做吗？（技能/时间/精力）
│
├─ Opportunity？
│   ├─ 有机会做吗？（渠道/触点）
│   └─ 环境支持吗？（政策/文化）
│
└─ Motivation？
    ├─ 想做吗？（意愿）
    └─ 为什么做？（内在/外在驱动）

优化方向：
C不足 → 降低能力要求、提供指导
O不足 → 增加触点、优化渠道
M不足 → 强化激励、情感共鸣
```

## 3. EAST 行为改变框架

### 四大原则
```
Easy（简单）+ Attractive（吸引）+ Social（社交）+ Timely（及时）
```

| 原则 | 策略 | 案例 |
|------|------|------|
| Easy | 默认选项、减少步骤 | 默认勾选、自动续费 |
| Attractive | 视觉突出、利益强调 | 按钮颜色、折扣标签 |
| Social | 社交证明、从众效应 | "已有X人购买" |
| Timely | 关键时刻提醒 | 购物车遗忘提醒 |

### 应用模板
```
【行为设计】目标行为：____

E（简单）：
- 当前步骤：__步
- 简化后：__步
- 简化措施：____

A（吸引）：
- 视觉设计：____
- 利益强调：____

S（社交）：
- 社交证明：____
- 从众暗示：____

T（及时）：
- 触发时机：____
- 提醒方式：____
```

## 4. Nudge 助推理论

### 核心定义
```
Nudge = 不限制选择、不改变激励，但影响决策的轻推

特点：
- 不禁止任何选项
- 不改变经济激励
- 避免非理性决策
```

### 常用助推类型

| 类型 | 方法 | 案例 |
|------|------|------|
| 默认效应 | 预设选项 | 默认勾选会员 |
| 锚定效应 | 设置参照点 | "原价X元"划线价 |
| 损失厌恶 | 强调损失 | "错过再等一年" |
| 稀缺效应 | 限时限量 | "仅剩3件" |
| 社会认同 | 展示他人行为 | "90%用户选择" |
| 选择架构 | 选项排序 | 推荐款置顶 |

### 助推设计原则
```
□ 是否保留用户自由选择权？
□ 是否基于真实行为洞察？
□ 是否可逆或可撤销？
□ 是否符合伦理（不欺骗）？
```

## 5. KANO 模型

### 需求层次分类
```
必备需求：没有不满意，有了无感
期望需求：满足程度与满意度成正比
魅力需求：没有无所谓，有了很惊喜
无差异需求：有无都不影响
反向需求：过度导致不满意
```

### 需求类型矩阵

| 类型 | 不提供 | 提供 | 优化策略 |
|------|--------|------|---------|
| 必备 | 不满意 | 基准 | 必须达到及格线 |
| 期望 | 不满意 → 满意 | 线性提升 | 持续优化 |
| 魅力 | 无感 | 惊喜 | 创造差异化 |
| 无差异 | 无感 | 无感 | 不投入 |
| 反向 | 满意 | 不满意 | 避免过度 |

### KANO 问卷法
```
问题设计（正反向各1题）：
1. 如果有这个功能，你感觉如何？
   □ 我喜欢 □ 理应如此 □ 无所谓 □ 勉强接受 □ 我不喜欢
2. 如果没有这个功能，你感觉如何？
   □ 我喜欢 □ 理应如此 □ 无所谓 □ 勉强接受 □ 我不喜欢

分类规则：对照KANO分类表
```

## 6. CJM 客户旅程地图

### 结构要素
```
阶段 + 行为 + 情绪 + 痛点 + 机会点
```

### 通用旅程阶段

| 阶段 | 用户目标 | 触点 | 情绪曲线 | 优化机会 |
|------|---------|------|---------|---------|
| 认知 | 了解产品 | 广告、内容 | 中立 | 曝光优化 |
| 考虑 | 评估选择 | 官网、客服 | 波动 | 内容完善 |
| 购买 | 完成交易 | 下单、支付 | 高峰/低谷 | 流程简化 |
| 使用 | 使用产品 | 产品、指南 | 波动 | 体验优化 |
| 忠诚 | 推荐复购 | 社群、服务 | 维持 | 运营活动 |

### CJM 绘制步骤
```
1. 定义用户画像
2. 梳理接触点（线上+线下）
3. 记录用户行为和情绪
4. 识别痛点和高光时刻
5. 标注机会点
6. 输出优化方案
```

### CJM 模板

## 7. 70+ 心理模型速查

### 认知偏差类

| 心理模型 | 定义 | 营销应用 |
|---------|------|---------|
| 锚定效应 | 依赖第一信息做判断 | 价格对比、原价划线 |
| 沉没成本 | 不愿放弃已投入 | 会员积分、预付费 |
| 损失厌恶 | 损失痛苦 > 收益快乐 | 限时优惠、试错机会 |
| 禀赋效应 | 拥有后估值更高 | 免费试用、退款宽限期 |
| 选择悖论 | 选项多反而难以决策 | 精选推荐、限制SKU |
| 心理账户 | 不同来源钱估值不同 | 奖金类促销、分开定价 |

### 社会影响类

| 心理模型 | 定义 | 营销应用 |
|---------|------|---------|
| 社会认同 | 参照他人行为 | 已购人数、热门标签 |
| 从众效应 | 跟随大流 | 畅销榜、热销标记 |
| 权威效应 | 信任权威 | KOL背书、专家推荐 |
| 互惠原则 | 接受后倾向回报 | 免费内容、赠品营销 |
| 稀缺性 | 越少越想拥有 | 限时、限量、仅剩N件 |

### 决策启发类

| 心理模型 | 定义 | 营销应用 |
|---------|------|---------|
| 峰终定律 | 印象由高峰和结尾决定 | 开箱体验、服务结尾 |
| 简单曝光效应 | 熟悉产生好感 | 重复曝光、品牌植入 |
| 对比效应 | 参照物影响感知 | 对比定价、高端定位 |
| 框架效应 | 表述方式影响决策 | 正向表述、"省多少" |
