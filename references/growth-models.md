# 用户增长模型

> 增长框架 / 用户旅程 / 留存变现 / 病毒裂变框架

---

## 1. AARRR 增长海盗模型

### 五层漏斗
```
Acquisition（获取）→ Activation（激活）→ Retention（留存）→ Revenue（变现）→ Referral（推荐）
```

| 阶段 | 核心指标 | 优化方向 |
|------|---------|---------|
| Acquisition | 新增用户数、获客成本（CAC） | 渠道优化、流量获取 |
| Activation | 激活率、核心功能使用率 | 首次体验优化、Aha Moment |
| Retention | 次日/7日/30日留存率 | 产品粘性、价值持续交付 |
| Revenue | 付费转化率、ARPU、LTV | 变现路径、定价策略 |
| Referral | 邀请率、K因子 | 裂变机制、分享激励 |

### 北极星指标映射
```
不同阶段北极星指标选择：
- 早期产品：留存率（验证PMF）
- 成长期：增长率（规模化获客）
- 成熟期：LTV/CAC比值（商业效率）
```

## 2. AIPL 模型（阿里）

### 消费者资产链路
```
Awareness（认知）→ Interest（兴趣）→ Purchase（购买）→ Loyalty（忠诚）
```

| 阶段 | 定义 | 阿里指标 | 营销动作 |
|------|------|---------|---------|
| A | 被品牌触达 | 曝光UV | 品牌广告、内容曝光 |
| I | 产生兴趣互动 | 点击、收藏、加购 | 内容种草、直播、优惠 |
| P | 完成购买 | 成交UV、GMV | 促销、转化优化 |
| L | 复购和忠诚 | 复购率、会员数 | 会员运营、私域 |

### AIPL 资产流转
```
A资产 → I资产转化率：点击率
I资产 → P资产转化率：成交转化率
P资产 → L资产转化率：复购率/会员转化率

健康度指标：
A:I:P:L ≈ 100:30:10:3（行业参考值）
```

## 3. AISAS 模型（电通）

### 互联网时代消费路径
```
Attention（注意）→ Interest（兴趣）→ Search（搜索）→ Action（行动）→ Share（分享）
```

| 阶段 | 用户行为 | 营销抓手 |
|------|---------|---------|
| A | 看到广告、内容 | 品牌曝光、KOL种草 |
| I | 产生兴趣、浏览 | 内容深度、视觉吸引 |
| S | 主动搜索信息 | SEO、口碑、评价 |
| A | 购买决策 | 转化优化、信任保障 |
| S | 分享体验 | 激励分享、UGC运营 |

### 与 AIDA 的关键差异
```
新增 Search（搜索）：用户主动权增强
新增 Share（分享）：消费后传播成为闭环

启示：
- 搜索布局成为关键（SEO/口碑）
- 用户体验决定二次传播
```

## 4. AIPS 模型（字节）

### 抖音电商路径
```
Awareness（认知）→ Interest（兴趣）→ Purchase（购买）→ Share（分享）
```

### 字节算法匹配
```
A → 兴趣推荐算法（曝光优化）
I → 内容互动算法（种草优化）
P → 转化预测算法（成交优化）
S → 社交推荐算法（裂变优化）

核心：算法驱动的内容分发 + 转化闭环
```

## 5. 北极星指标

### 定义与选择原则
```
北极星指标 = 反映核心价值交付的唯一指标

选择标准：
1. 衡量用户价值，而非商业利益
2. 反映产品核心功能使用
3. 可拆解到各团队执行
4. 长期导向，非短期数字
```

### 各行业北极星示例

| 公司 | 北极星指标 | 理由 |
|------|-----------|------|
| Spotify | 听歌时长 | 核心价值是音乐消费 |
| Airbnb | 预订间夜数 | 核心价值是住宿交易 |
| Slack | 发送消息数 | 核心价值是团队沟通 |
| Zoom | 会议分钟数 | 核心价值是视频通话 |
| Netflix | 观看时长 | 核心价值是内容消费 |

### 指标拆解树
```
北极星指标
    ├─ 一级指标（部门KPI）
    │   ├─ 二级指标（团队指标）
    │   │   └─ 三级指标（个人指标）
    │   └─ ...
    └─ ...

示例（电商）：
GMV（北极星）
├─ 流量
│   ├─ 新客流量
│   └─ 老客流量
├─ 转化率
│   ├─ 首购转化
│   └─ 复购转化
└─ 客单价
    ├─ 单价
    └─ 购买件数
```

## 6. 增长飞轮

### 亚马逊飞轮
```
低价 → 更多客户 → 更多卖家 → 更丰富选择 → 更好体验 → 更多客户

核心：良性循环、自我强化
```

### 飞轮设计三要素
```
1. 驱动力：什么启动飞轮？
2. 正反馈：什么加速飞轮？
3. 飞轮半径：涉及多少环节？

设计检验：
□ 飞轮是否可启动？
□ 各环节是否真实连接？
□ 是否有外部动力维持？
```

### 常见飞轮类型
| 类型 | 案例 | 核心逻辑 |
|------|------|---------|
| 平台飞轮 | Amazon、淘宝 | 供需双边网络效应 |
| 内容飞轮 | TikTok、小红书 | 内容→流量→创作者 |
| 社交飞轮 | 微信、Facebook | 用户→社交网络→新用户 |
| 数据飞轮 | Google、AI产品 | 数据→模型→体验→用户→数据 |

## 7. Hook 模型

### 上瘾机制
```
Trigger（触发）→ Action（行动）→ Variable Reward（多变酬赏）→ Investment（投入）
```

| 阶段 | 设计要点 | 案例 |
|------|---------|------|
| Trigger | 外部触发（推送）+ 内部触发（情绪） | 无聊时刷抖音 |
| Action | 降低行动门槛、简化流程 | 下拉即刷新 |
| Variable Reward | 不确定性奖励、期待感 | 刷到下一个未知视频 |
| Investment | 用户投入增加留存 | 关注、点赞、收藏 |

### Hook 检验清单
```
□ 是否有明确的外部触发机制？
□ 用户内部触发是什么情绪/场景？
□ 行动是否足够简单？
□ 奖励是否有变化和期待感？
□ 用户投入是否能增加产品价值？
```

## 8. 裂变增长模型

### 裂变核心公式
```
病毒系数 K = i × c
i = 每个用户发出的邀请数
c = 邀请转化率

K > 1 → 病毒式增长
K = 1 → 稳定增长
K < 1 → 需持续获客投入
```

### 裂变类型矩阵

| 类型 | 驱动力 | 案例 | K值范围 |
|------|--------|------|---------|
| 利益裂变 | 优惠券、红包 | 拼多多、瑞幸 | 1-2 |
| 功能裂变 | 功能协同 | Zoom、石墨 | 0.5-1.5 |
| 社交裂变 | 社交货币 | 小红书、朋友圈 | 0.8-2 |
| 内容裂变 | 内容价值 | 抖音、公众号 | 0.5-1.5 |

### 裂变设计框架
```
1. 确定裂变目标（拉新/促活/转化）
2. 设计裂变诱饵（利益/功能/社交/内容）
3. 降低分享门槛（一键转发、邀请码）
4. 优化转化路径（落地页、新人礼包）
5. 数据追踪优化（K值、渠道ROI）
```

## 9. 人群漏斗模型

### 通用漏斗结构
```
曝光人群 → 点击人群 → 访问人群 → 咨询人群 → 购买人群 → 复购人群
```

### 漏斗效率诊断
```
整体转化率 = 各层转化率相乘

优化优先级：
先优化转化率最低的环节（瓶颈）
再优化流量最大的环节（杠杆）

漏斗健康度：
层间转化率 > 行业基准 → 健康
层间转化率 < 行业基准 → 优化重点
```
