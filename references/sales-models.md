## 1. 销售漏斗

### 标准销售漏斗
```
线索（Leads）→ 合格线索（MQL）→ 销售线索（SQL）→ 商机（Opportunity）→ 成交（Closed Won）
```

| 阶段 | 定义 | 关键动作 | 转化率基准 |
|------|------|---------|-----------|
| Leads | 任何联系方式 | 大量获取 | 100% |
| MQL | 营销合格线索 | 评分筛选 | 20-30% |
| SQL | 销售合格线索 | 需求确认 | 30-50% |
| Opportunity | 明确商机 | 方案/报价 | 25-40% |
| Closed Won | 成交 | 签约/付款 | 15-25% |

### 漏斗效率诊断
```
线索 → 成交整体转化率 = 各层转化率相乘
行业基准：2-5%

优化方向：
- 线索量不足 → 加大获客投入
- MQL→SQL转化低 → 优化线索质量
- SQL→商机转化低 → 提升销售能力
- 商机→成交转化低 → 优化方案/定价
```

## 3. 定价策略

### 定价方法矩阵

| 方法 | 逻辑 | 适用场景 | 风险 |
|------|------|---------|------|
| 成本加成 | 成本+利润率 | 传统行业、标准化 | 忽略客户价值 |
| 竞争导向 | 参照竞品定价 | 红海市场 | 价格战 |
| 价值导向 | 客户感知价值 | 差异化产品 | 价值难以量化 |
| 动态定价 | 需求波动定价 | 航旅、电商 | 用户体验问题 |

### 心理定价技巧

| 技巧 | 方法 | 效果 |
|------|------|------|
| 魅力定价 | ¥99 而非 ¥100 | 感知更便宜 |
| 锚定定价 | 先展示高价再展示实价 | 感知划算 |
| 捆绑定价 | 套餐 > 单品 | 提升客单价 |
| 分层定价 | 基础/标准/高级 | 引导中档选择 |
| 声望定价 | 高价=高品质 | 品牌溢价 |

### 定价实验框架
```
Step 1：确定价格弹性区间
Step 2：设计价格测试方案（A/B）
Step 3：选择测试样本
Step 4：运行实验（1-2周）
Step 5：分析收入最大化价格点

核心公式：
最优价格 = argmax[P × Q(P)]
P = 价格，Q(P) = 价格P时的需求量
```

## 4. 客户成功模型

### 核心公式
```
客户成功 = 客户实现预期结果 + 体验过程顺畅

CSM 目标：
- 提升留存率（续约率）
- 增加扩展收入（增购/交叉销售）
- 降低流失率
- 提升 NPS
```

### 客户健康度评分
```
健康度 = f(使用深度 + 互动频率 + 满意度 + 商业指标)

评分维度：
| 维度 | 指标 | 权重 |
|------|------|------|
| 使用深度 | 核心功能使用率 | 30% |
| 互动频率 | 登录频次 | 20% |
| 满意度 | NPS/CSAT | 25% |
| 商业指标 | 续约可能性 | 25% |

分级：
> 80分：健康（绿色）
60-80分：关注（黄色）
< 60分：风险（红色）
```

### 客户生命周期管理
```
Onboarding（入驻期）：
- 目标：快速实现首次价值
- 动作：引导培训、首次成功
- 周期：1-30天

Adoption（采用期）：
- 目标：深度使用核心功能
- 动作：功能引导、最佳实践
- 周期：30-90天

Expansion（扩展期）：
- 目标：增加使用广度和深度
- 动作：交叉销售、增值服务
- 周期：90-365天

Renewal（续约期）：
- 目标：成功续约
- 动作：价值回顾、续约谈判
- 周期：合同到期前90天

Advocacy（推荐期）：
- 目标：转化为推荐者
- 动作：推荐激励、案例合作
- 周期：持续
```

## 5. 2B 销售流程

### 标准销售阶段
```
1. 线索获取 → 2. 需求发现 → 3. 方案设计 → 4. 商务谈判 → 5. 签约交付
```

| 阶段 | 关键动作 | 产出物 | 胜率 |
|------|---------|--------|------|
| 线索获取 | 渠道获取、评分 | 合格线索 | 100% |
| 需求发现 | 痛点挖掘、决策链 | 需求文档 | 70% |
| 方案设计 | 方案编写、演示 | 解决方案 | 50% |
| 商务谈判 | 报价、商务条款 | 合同草案 | 35% |
| 签约交付 | 签约、交接 | 合同+交接 | 25% |

### 关键指标
```
销售周期：从线索到成交的平均时间
赢单率：商机到成交的转化率
平均客单价：ACV/TCV
销售效率：新增ARR / 销售成本
```

## 6. 变现模式矩阵

### 常见变现模式

| 模式 | 逻辑 | 适用 | 优势 | 劣势 |
|------|------|------|------|------|
| 订阅制 | 周期性付费 | SaaS、内容 | 可预测收入 | 获客成本高 |
| 交易抽成 | 按交易收费 | 平台、电商 | 规模效应 | 依赖交易量 |
| 广告变现 | 流量售卖 | 媒体、社交 | 边际成本低 | 用户体验冲突 |
| 增值服务 | 基础免费+付费 | 工具、游戏 | 试用门槛低 | 转化率低 |
| 授权许可 | 一次性买断 | 软件、IP | 回款快 | 无持续收入 |

### 变现组合策略
```
单一变现 → 组合变现 → 生态变现

示例（内容平台）：
单一：广告
组合：广告 + 会员 + 电商
生态：广告 + 会员 + 电商 + 知识付费 + 品牌合作

选择原则：
1. 不伤害核心用户体验
2. 每个模式有独立增长逻辑
3. 模式间有协同效应
4. 变现效率递增
```
