# 案例演示：小红书种草传播

> 场景：一个国产无钢圈内衣品牌，想在小红书做种草传播。

---

## 应用框架

按「场景四：内容平台种草传播」路径：

```
Step 1: 于冬琪四步循环 → Step 2: 林桂枝文案技巧 → Step 3: 关健明转化优化 → Step 4: 彭芳私域承接
```

---

## Step 1: 于冬琪 捕捉 → 理解 → 放大 → 激发

### 捕捉

小红书搜索分析（模拟数据）：

| 用户搜什么 | 搜索量 | 竞争度 |
|-----------|--------|--------|
| 无钢圈内衣推荐 | 极高 | 高 |
| 大胸显小内衣 | 高 | 中 |
| 小胸不空杯 | 高 | 中 |
| 内衣勒痕怎么办 | 中 | 低 ← 机会！ |
| 夏天内衣不透气 | 高 | 中 |

用户在评论区问什么：
- "真的不勒吗？我穿一整天回家脱掉内衣，印子要半小时才消"
- "大胸穿无钢圈会不会下垂？"
- "出汗之后会不会有味道？"

### 理解

**三层理解**：

| 层 | 内容 |
|----|------|
| 第一层（表面） | "想要一件舒服的内衣" |
| 第二层（深层） | "想要一件穿一整天都不难受、回家脱掉没有红印子、不用担心走光的内衣" |
| 第三层（动机） | "我每天已经够累了——工作、带娃、通勤。内衣不应该再给我增加不舒服。我值得被好好对待，从最贴身的那件衣服开始。" |

### 放大

内容矩阵：

| 内容角度 | 核心信息 | KOC选择 |
|---------|---------|---------|
| 功课型 | "内衣怎么挑？看这3个细节" | 穿搭博主 |
| 实测型 | "穿了8小时的无钢圈内衣实录" | 素人KOC |
| 场景型 | "上班/带娃/运动/睡觉，一件搞定" | 职场妈妈KOC |
| 对比型 | "XX vs XX，无钢圈内衣终极PK" | 测评博主 |
| 故事型 | "从每天回家第一件事脱内衣，到忘了它的存在" | 情感类KOC |

### 激发

- 内容中植入"搜 XX 内衣"引导
- 品牌搜索页承接：品牌故事 + 主推款 + 好评精选
- 首单优惠：新客立减30元

---

## Step 2: 林桂枝 文案技巧应用

### 社交媒体标题（应用27条法则）

```
功课型：
"内衣怎么挑？看这3个细节就够了（第3个90%的人不知道）"

实测型：
"穿了8小时的无钢圈内衣，脱掉的那一刻我愣住了"

心语型：
"那些一回家就迫不及待脱内衣的姐妹，懂的都懂"
```

### 视频文案（应用24招）

3秒开头：
> "你每天回家第一件事是什么？我的答案是……脱内衣。"

---

## Step 3: 关健明 转化优化

在品牌搜索承接页应用4步法：

- **激发欲望**：感官占领——"穿上之后，就像没有穿一样"
- **赢得信任**：事实证明——透气度测试、弹力测试、100位用户8小时试穿报告
- **引导下单**：价格锚点——"同品质进口品牌定价300+，我们只要129"

---

## Step 4: 彭芳 私域承接

### 私域路径

```
小红书种草（公域）
  → 评论/私信引导加微信
    → 朋友圈日常内容（生活+价值输出）
      → 拉群（内衣穿搭分享群）
        → 群内会员日/新品首发/限时秒杀
```

### 朋友圈运营

| 内容 | 示例 |
|------|------|
| 生活化 | 办公室日常、产品开发过程 |
| 价值输出 | 内衣保养小技巧、不同身材怎么选内衣 |
| 互动 | "你最在意内衣的什么？投票：A.舒服 B.好看 C.价格" |
| 产品 | 新品剧透、用户好评截图、限时活动 |
