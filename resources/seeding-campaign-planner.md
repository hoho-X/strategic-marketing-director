### 2.3 种草内容角度提炼

基于对用户深层需求的理解，提炼可被种草的内容角度：

1. **功课型**：帮用户"做功课"对比分析
2. **体验型**：真实使用体验分享
3. **场景型**：特定场景下的解决方案
4. **方法型**：使用技巧/搭配方法
5. **故事型**：使用前后的变化故事

## Step 3: 放大（让真实体验被看见）

### 3.1 内容矩阵规划

| 内容类型 | KOC/KOL层级 | 数量 | 核心角度 | 发布时间 |
|---------|-----------|------|---------|---------|
| 素人笔记 | 100-1K粉 | | 真实体验 | |
| KOC | 1K-10K粉 | | 深度功课 | |
| 腰部KOL | 10K-100K粉 | | 专业解读 | |
| 头部KOL | 100K+粉 | | 趋势引领 | |

### 3.2 内容自检清单

每条种草内容发布前检查：

- [ ] 像广告吗？（太像→重写，种草不像广告）
- [ ] 有用吗？（能帮用户做功课/决策吗？）
- [ ] 真实吗？（有具体的场景和细节吗？）
- [ ] 有情绪吗？（看了之后有感觉吗？）
- [ ] 能让用户"看到自己"吗？（"这不就是我吗"的感觉）

### 3.3 放大策略

| 策略 | 做法 | 适用 |
|------|------|------|
| 矩阵铺量 | 同角度多账号同时发 | 需要快速建立认知 |
| 话题引爆 | 一个爆款带动UGC | 有强社交属性的产品 |
| 场景渗透 | 每个场景逐一击破 | 多场景多用途产品 |
| 人格绑定 | 一个标志性KOL长期合作 | 品牌人格化需求强 |

## Step 4: 激发（转化为购买）

### 4.1 转化路径设计

```
看到种草内容
    → 搜索品牌/产品 → 看到更多内容 → 进入购买渠道
    → 直接评论/私信问 → 引导到购买渠道
    → 收藏/点赞 → 后续触达 → 购买
```

### 4.2 转化钩子设计

| 钩子类型 | 怎么做 |
|---------|--------|
| 搜索引导 | 在内容中引导用户搜索品牌词 |
| 评论区引导 | 在评论区回复中引流到店铺/直播间 |
| 私域承接 | 引导加微信/进群，后续运营转化 |
| 限时活动 | 配合平台大促/品牌活动日集中转化 |

### 4.3 效果衡量

| 种草心智率（看到→形成认知） | | | |

## 周期规划

| 阶段 | 时间 | 重点 |
|------|------|------|
| 预热期 | 前1-2周 | 铺量建立基础内容矩阵 |
| 引爆期 | 第3-4周 | 集中发力、话题引爆 |
| 转化期 | 第5-6周 | 配合大促/活动收割 |
| 沉淀期 | 持续 | UGC运营、品牌词积累 |

---

## 关键提醒

- 种草不是一次活动，是长期能力建设
- 产品本身必须"值得被种草"——这是前提
- 用户真实体验 > 精致内容，不要过度包装
- 种草需要全公司协同——产品、市场、销售、客服都要参与
