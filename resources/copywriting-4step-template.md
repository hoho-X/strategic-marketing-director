# 爆款文案写作模板

> 基于关健明《爆款文案》4步18法，填空即用的销售文案模板。

---

## 模板结构

```
[标题] ← 选一种标题写法
[欲望段] ← 选2-3种欲望激发方法
[信任段] ← 选1-2种信任建立方法
[下单段] ← 选2种下单引导方法
```

---

## 第一步：标题（5选1）

### 选题1：新闻社论
```
【刚刚/最新/重磅】[发生的变化/事件]
[这个变化对你的影响]
```

### 选题2：好友对话
```
[称呼]，[告诉你一个和你有关的信息]
```

### 选题3：实用锦囊
```
[数字]个方法，帮你[解决什么问题]，[第几个]最[形容词]
```

### 选题4：惊喜优惠
```
[平时多少钱/多难买]，今天[特殊价/特殊福利]，[为什么现在买]
```

### 选题5：意外故事
```
[反常识的事实/过程]，结果居然[戏剧性的结果]
```

---

## 第二步：激发欲望（6选2-3）

### 方法1：感官占领
```
当你[使用产品的场景]：
👀 你会看到——
👂 你会听到——
👃 你会闻到——
🤚 你会感觉到——
```

### 方法2：恐惧诉求
```
想象一下这个场景：[痛苦场景]

如果[什么都不做]，后果是：[严重后果]

但如果你[使用产品]，结果就完全不同：[美好结果]
```

### 方法3：认知对比
```
[旧方案/竞品/不作为]：[缺点1]、[缺点2]、[缺点3]

而[我们的产品]：[优点1]、[优点2]、[优点3]
```

### 方法4：使用场景
```
[时间]，[地点]
你正在[做什么]
这时候[痛点/困扰]
[产品]让你[爽点/结果]

比如：
· 早上7点，赶地铁——用[产品]，[好处]
· 中午12点，办公室午休——用[产品]，[好处]
· 晚上10点，准备睡觉——用[产品]，[好处]
```

### 方法5：畅销
```
[产品]已经：
· 累计销量：[数字]
· 用户数：[数字]
· [名人/达人]同款
· 上个月：[数字]人购买
· 好评率：[数字]
```

### 方法6：顾客证言
```
@[用户名]：[具体场景]，用了[时间]，[具体效果]

@[用户名]：[对比之前的困扰]，现在[改变]

@[用户名]：本来是[担心]，结果[超出预期的体验]
```

---

## 第三步：赢得信任（3选1-2）

### 方法1：权威转嫁
```
[专家/机构/奖项] 认可：
· [机构名]认证：[具体认证内容]
· [专家名]推荐："[推荐语]"
· 获得[奖项名称]
· 服务过[知名品牌/客户]
```

### 方法2：事实证明
```
[产品] 的数据：
· [可验证数据1]
· [可验证数据2]
· [可验证数据3]

生产过程：
· 原料来自[产地]
· 采用[工艺/技术]
· 经过[检测/认证]

（附：检测报告编号 / 证书 / 工厂实拍）
```

### 方法3：化解顾虑
```
你可能担心：
❓ [顾虑1] → ✅ [我们的方案]
❓ [顾虑2] → ✅ [我们的方案]
❓ [顾虑3] → ✅ [我们的方案]
```

---

## 第四步：引导下单（4选2）

### 方法1：价格锚点
```
[同类/更高端产品]：[高价]
[我们的产品]，同样的[核心价值]，只要[低价]

相当于用[比例]的价格，享受[同等的价值]
```

### 方法2：算账
```
[产品]价格是[总价]

但如果你算一算：
用[时间长度]，每天只要[单价]
= [日常小花费对比]

[日常小花费] vs [产品带来的长期价值]
你选哪个？
```

### 方法3：正当消费
```
你买[产品]不是在花钱，是在：
✅ [正向意义1]
✅ [正向意义2]
✅ [正向意义3]
```

### 方法4：限时限量
```
⏰ [截止时间] 前下单，享受[优惠]
📦 仅剩[数量]份
🔥 [前XX名]加赠[赠品]
```

## 写完后自检清单

- [ ] 标题：读完想点开吗？（不想→重写标题）
- [ ] 欲望段：读完想要这个产品吗？（不想→加强感官占领/恐惧诉求）
- [ ] 信任段：读完后相信产品有效吗？（不信→加事实证明/权威转嫁）
- [ ] 下单段：读完想立刻买吗？（不想→加限时限量/强化算账）
- [ ] 整体：像广告吗？（太像→用林桂枝的方法，改得更像人话）
- [ ] 整体：字数可以再删一半吗？（试试看）
