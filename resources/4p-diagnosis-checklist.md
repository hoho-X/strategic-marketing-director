# 4P 营销诊断清单

> 基于麦卡锡4P框架 + 小马宋经营诊断思维，用于快速检查一个品牌/产品的营销系统健康度。

---

## 使用说明

逐P逐项打分（✅ 健康 / ⚠️ 有风险 / ❌ 严重问题），最终汇总判断。

---

## Product（产品）

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| P1 | 产品解决了用户的什么真实问题？ | | |
| P2 | 产品和竞品有可感知的差异化吗？ | | |
| P3 | 用户会因为这个差异化多付钱吗？ | | |
| P4 | 产品品质和定价匹配吗？ | | |
| P5 | 产品有让用户自发推荐的理由吗？ | | |
| P6 | 产品线宽度是否合理？（太多/太少？） | | |
| P7 | 产品包装/外观与品牌调性一致吗？ | | |
| P8 | 配套服务（售后/客服/使用指导）够吗？ | | |

## Price（价格）

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| R1 | 定价和战略定位一致吗？（成本领先≠高价） | | |
| R2 | 目标客群觉得这个价格"值"吗？ | | |
| R3 | 和直接竞品相比，价格在什么位置？合理吗？ | | |
| R4 | 价格是否能支撑盈利？ | | |
| R5 | 渠道佣金/折扣结构合理吗？ | | |
| R6 | 价格是否给消费者提供了心理安全感？（太便宜有时是劣势） | | |
| R7 | 消费者决策时是否在意价格？（敏感/不敏感） | | |

## Place（渠道）

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| C1 | 目标消费者能方便地买到吗？ | | |
| C2 | 渠道档次和品牌定位一致吗？ | | |
| C3 | 线上线下是否有冲突？（价格/体验不一致） | | |
| C4 | 渠道成本占比合理吗？ | | |
| C5 | 终端体验（陈列/展示/店员话术）统一吗？ | | |
| C6 | 库存管理是否健康？（不断货/不积压） | | |
| C7 | 有没有被渠道卡脖子的风险？ | | |

## Promotion（推广）

| # | 检查项 | 状态 | 备注 |
|---|--------|------|------|
| M1 | 目标消费者通过什么渠道获取信息？你覆盖了吗？ | | |
| M2 | 推广内容传递的核心信息是什么？清晰吗？ | | |
| M3 | 推广投入产出比（ROI）合理吗？ | | |
| M4 | 品牌声量和销量是增长还是衰退？ | | |
| M5 | 用户口碑/NPS是怎样的？ | | |
| M6 | 竞品在推广上做什么？我们是被动跟随还是主动引领？ | | |
| M7 | 推广内容和品牌定位一致吗？（不要高档品牌做低价促销） | | |

---

## 4P一致性检查（最关键）

| 检查项 | 状态 | 问题描述 |
|--------|------|---------|
| 产品档次 ↔ 价格水平 是否一致？ | | |
| 产品档次 ↔ 渠道档次 是否一致？ | | |
| 产品档次 ↔ 推广调性 是否一致？ | | |
| 价格水平 ↔ 渠道类型 是否一致？ | | |
| 价格水平 ↔ 推广内容 是否一致？ | | |
| 渠道类型 ↔ 推广方式 是否一致？ | | |

> ⚠️ 如果上面任何一项出现「不一致」，这个不一致可能就是品牌问题的根源。
