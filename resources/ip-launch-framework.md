# IP 发售框架

> 基于彭芳《引爆》发售体系，整合文案+私域+发售三位一体方法论。

---

## 核心公式

```
发售成功 = 文案力 × 私域力 × 发售力
```

一项为零，结果为零。三项都必须到位。

---

## 模块一：文案力（内容弹药）

### 1.1 自检

你的文案是自嗨还是走心？

| 自嗨文案的特征 | 走心文案的特征 |
|-------------|-------------|
| "我们采用了XX技术" | "你再也不用为XX烦恼了" |
| 功能参数罗列 | 使用场景描述 |
| 没有真实案例 | 有具体客户故事 |
| 读起来像说明书 | 读起来像朋友聊天 |

### 1.2 五种发售文案类型

| 文案类型 | 目的 | 发布时间 |
|---------|------|---------|
| **痛点文** | 唤起需求——"你是否有过这样的困扰" | 预热期前3天 |
| **价值文** | 展示方案——"这就是解决方案" | 预热期中段 |
| **信任文** | 建立信心——"为什么相信我" | 预热期末尾 |
| **成交文** | 临门一脚——"现在下单的N个理由" | 发售日 |
| **追销文** | 善后收割——"错过了？还有机会" | 发售日后 |

### 1.3 13字心法（文案脱胎换骨的秘诀）

> 彭芳的核心独门秘籍，详见原著《引爆》。

### 1.4 5个戒除自嗨的方法即时版

1. 把"我们"换成"你"
2. 把"卖点"换成"痛点解决后的画面"
3. 把"等等"换成"具体数字"
4. 把"参数"换成"感受"
5. 把"道理"换成"客户故事"

## 模块二：私域力（信任基建）

### 2.1 私域布局

| 阵地 | 作用 | 运营重点 |
|------|------|---------|
| 个人微信号 | 最深度的连接 | 朋友圈内容、1v1沟通 |
| 微信群 | 批量触达+社群氛围 | 群公告、群活动、群接龙 |
| 公众号 | 内容沉淀 | 长文深度输出 |
| 视频号/直播 | 实时互动 | 发售日集中转化 |

### 2.2 朋友圈运营节奏

| 内容类型 | 占比 | 示例 |
|---------|------|------|
| **生活化内容** | 30% | 日常分享、个人思考 |
| **价值输出** | 40% | 行业观点、使用技巧、客户案例 |
| **互动话题** | 20% | 提问、投票、征集 |
| **产品/发售** | 10% | 产品展示、发售预告、战报 |

### 2.3 信任积累时间线

```
第1周 → 价值输出为主，不推销
第2周 → 开始分享客户案例
第3周 → 透露"正在做一个新产品/服务"
第4周 → 正式发售
发售中 → 实时战报 + 客户证言 + 限时限量
```

## 模块三：发售力（成交引擎）

### 3.1 发售三阶段

#### 预热期（3-7天）

| 动作 | 内容 | 检查 |
|------|------|------|
| 悬念设置 | "我最近在做一件事..." | [ ] |
| 价值预告 | "会帮你解决[具体问题]" | [ ] |
| 用户痛点共鸣 | "你是不是也遇到..." | [ ] |
| 案例剧透 | "已经帮[某客户]做到了[某结果]" | [ ] |
| 预约引导 | "扫码进群/预约，锁定首批名额" | [ ] |
| 倒计时 | "倒计时X天" | [ ] |

#### 发售期（1-3天）

| 动作 | 内容 | 检查 |
|------|------|------|
| 产品完整展示 | 是什么/解决什么/和别人的区别 | [ ] |
| 价值最大化 | 不止功能，还有情感/身份价值 | [ ] |
| 信任背书集中展示 | 资质、案例、证言、数据 | [ ] |
| 超值赠品/权益 | 限时附赠，制造"错过亏大了" | [ ] |
| 限时限量 | 截止时间/名额限制 | [ ] |
| 实时战报 | "已有XX人下单" | [ ] |
| 客户证言穿插 | 真实付款截图/好评 | [ ] |
| 答疑解惑 | 集中回答常见问题 | [ ] |
| 涨价预告 | "X小时后恢复原价" | [ ] |

#### 追销期（1-3天）

| 动作 | 内容 | 检查 |
|------|------|------|
| 未成交私聊跟进 | 一对一了解犹豫原因 | [ ] |
| "最后机会"通知 | 限时返场/少量余位 | [ ] |
| 已成交用户交付 | 产品/服务正式开始交付 | [ ] |
| 案例素材收集 | 收集用户反馈，为下轮储备 | [ ] |
| 复盘总结 | 数据复盘、流程优化 | [ ] |

### 3.2 五大价值塑造秘籍应用

在发售过程中的每个环节，检查是否用到了五：

| 秘籍 | 在哪个环节用 | 怎么用 |
|------|------------|--------|
| 稀缺性 | 发售期 | 限量名额/限时价格 |
| 权威性 | 预热期+信任文 | 你的资历/成果/背书展示 |
| 从众性 | 发售期 | 实时战报/"XX人在看/在买" |
| 互惠性 | 预热期 | 先提供免费价值，再求成交 |
| 一致性 | 全流程 | 从小小的"预约""进群"开始 |

## 发售节奏模板（7天版）

| 天数 | 阶段 | 核心动作 |
|------|------|---------|
| Day 1 | 预热 | 痛点文：唤起需求 |
| Day 2 | 预热 | 价值文：展示方向 |
| Day 3 | 预热 | 案例文：证实效果 |
| Day 4 | 预热 | 信任文：为什么是你 |
| Day 5 | 发售 | 完整产品展示 + 下单引导 |
| Day 6 | 发售 | 战报 + 客户证言 + 最后机会 |
| Day 7 | 追销 | 私聊跟进 + 余位通知 |

## 成功后复盘

| 复盘维度 | 问题 |
|---------|------|
| 数据 | 预约量/进群量/下单量/转化率/客单价？ |
| 文案 | 哪篇文案效果最好？为什么？ |
| 节奏 | 时间节奏是否合适？太快/太慢？ |
| 信任 | 用户最大的下单犹豫是什么？下次怎么提前化解？ |
| 素材 | 收集了哪些可复用的案例/证言/数据？ |
