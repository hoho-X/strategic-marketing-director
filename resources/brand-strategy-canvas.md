## 分步填写指南

### 1. 商业课题

**填写**：一句话说清楚我们要解决什么问题

**注意**：不是"我们要做一次营销活动"，而是"我们面临什么商业问题需要品牌来解决"

**示例**：
- ❌ "我们要提高知名度" → 太模糊
- ✅ "新品牌进入市场3个月，目标客群不知道我们的存在" → 具体可衡量

**生意来源判断**：
- 新消费者：用户没买过这类产品
- 增加消耗：买了但用得不够多/不够频
- 抢份额：用户现在买的是竞品

### 2. 五维分析

**企业**：
- 真正想要什么：______
- 擅长什么：______
- 不能做什么：______

**品类**：
- 属于什么品类（消费者视角）：______
- 品类阶段：□ 导入 □ 成长 □ 成熟 □ 衰退
- 品类趋势：______

**竞争**：
- 直接竞品（消费者对比的品牌）：
  1. ______ 占据了心智词：______
  2. ______ 占据了心智词：______
  3. ______ 占据了心智词：______
- 间接竞品（替代方案）：______

**消费者**：
- 最有意义的潜在用户：______
- 他们的核心需求（功能/情感/社交）：______
- 他们对现有方案最不满意的地方：______
- 关键使用场景：______

**渠道**：
- 用户发现 → 了解 → 购买 → 复购的路径：______
- 渠道中的关键时刻：______

### 3. 核心洞察

**洞察的判断标准**（叶明桂版）：
- ✅ 是心理活动（不是行为表现）
- ✅ 是感性共鸣（不是理性判断）
- ✅ 是剖析（不是描述）

**洞察挖掘方法**：
连续问 5 个「为什么」

| 问 | 示例 |
|----|------|
| 用户买了什么？ | 买了高端矿泉水 |
| 为什么买？ | 因为觉得健康 |
| 为什么觉得这个健康？ | 因为看到广告说"来自阿尔卑斯山" |
| 为什么"来自阿尔卑斯山"=健康？ | 因为在想象中，那是纯净的、远离工业污染的地方 |
| **洞察** | 用户买的不是水，是"未被污染的纯净"的心理安全感 |

### 4. 品牌基因（品牌拟人化）

把品牌当一个人来定义：

```
如果品牌是一个人——

性别/年龄：  __________________
性格（3个形容词）： __________________
价值观：     __________________
说话方式：   __________________
他/她不会做的事： __________________
最打动人的特质： __________________
```

### 5. 出圈策略（5选1-3）

| 策略 | 你的品牌适用吗？ | 怎么做？ |
|------|---------------|---------|
| 独特消费场景 | 有"不常见但关键"的场景 | 定义场景 → 强化品牌在该场景的唯一性 |
| 差异需求 | 竞品是冷冰冰的大品牌 | 品牌拟人化 → 塑造更有温度的形象 |
| 精神制高点 | 品类有精神层次可占领 | 找到功能利益之上的精神终极利益 |
| 社会价值冲突 | 品牌价值观能引发共鸣 | 找到社会张力 → 用品牌故事表达立场 |
| 统一场景动力 | 多产品线需要统一品牌 | 提炼一个贯穿所有产品的品牌主张 |

### 6. 品牌主张

**一句话表达品牌的核心理念**（外部传播用）

格式建议：[品牌名]，[动词/形容词] [价值/态度]

示例：
- 耐克：Just Do It
- 苹果：Think Different
- 小红书：标记我的生活

### 7. 仪式感设计

**购买仪式**：用户购买时的独特体验是什么？
（开箱/到店/下单过程的特别设计）

**使用仪式**：使用产品时有什么固定动作/习惯？
（泡面一定要等3分钟、喝红酒先醒酒）

**社交仪式**：用户怎么分享/展示/交流这个品牌？
（打卡拍照角度、特定的分享话术）

## 完成后检查

- [ ] 品牌基因是否"有血有肉"（不是空洞形容词）？
- [ ] 洞察是否通过了"是心理活动/感性共鸣/剖析"三位检验？
- [ ] 出圈策略和品牌基因是否一致？
- [ ] 品牌主张是否能一句话让人记住？
- [ ] 4P是否与品牌策略一致？（用《4P诊断清单》交叉检查）
